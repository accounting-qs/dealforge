# Fallback Format Rules — Calendar Blocker Description (9-Part Structure)

Use this structure when the Deal Forge format brain is unavailable at runtime.
Max 300 words total. Each part is one tight sentence or short paragraph.

1. CONDITIONAL OPENER — "If you run [segment] and your growth still depends on [painful
   current state] — this will fix it." Qualifies the reader, diagnoses their pain, makes
   a bold promise. Frame as invitation to self-identify, not a declaration of pain.

2. CLIENT PROOF STORY — One client. Segment-matched. Specific before/after numbers.
   Structure: [Name] → [before state] → [used the system] → [result with verbatim numbers].
   Never fabricate numbers. If no case study, skip to Part 3.

3. CONTRAST FRAME — ADAPTS to host business_model.
   • Educational hosts (consulting / coaching / agency): "Instead of [what they stopped
     doing], [name] built [new system] — [what that system does]." Names old vs. new.
   • Delivery hosts (managed_service / lead_gen / saas / platform / product / unknown):
     "Instead of [running X yourself / building it in-house], [host] delivers [outcome]
     for you." The attendee does NOT build the new thing — the host does.

4. SESSION PROMISE — ADAPTS to host business_model.
   • Educational: "[Prospect company] will show you exactly how [they/client name] did
     this." Commits to a teachable mechanism.
   • Delivery: "[Prospect company] delivered [outcome] for [segment] — here's how that
     same engine can produce results for you." OR for SaaS/Product: "[Prospect company]
     will show you what's possible inside [product] for [segment]."
   • Never a pitch. Never frame the attendee as someone who will "build" anything when
     business_model is delivery.

5. YES / MAYBE / UNSUBSCRIBE / REGISTRATION LINK (non-negotiable) — Comes BEFORE bullets.
   Include all three elements:
   "Click 'YES' or 'MAYBE' to confirm your attendance. Click Unsubscribe and we won't
   reach out again. Or visit the official registration page to sign up there."
   YES/MAYBE = low-friction conversion (one click, no form). Unsubscribe = compliance +
   trust signal. Registration page = second pathway for standard funnel tracking.

6. DISCOVERY BULLETS — 4–5 bullets. Each is a SPECIFIC PROMISE, not a topic.
   The verb set adapts to host business_model:
   • Educational hosts → TRANSFORMATION promises the attendee will achieve.
     Wrong: "Webinar structure" (topic). Right: "Structure your webinar so prospects show
     up already convinced and ready to buy" (transformation). Verbs: Build, Structure,
     Master, Create, Develop, Implement.
   • Delivery hosts → OUTCOME promises the host delivers; attendee just receives.
     Wrong: "Build a lead generation system" (delivery host attendee doesn't build).
     Right: "Receive 50+ qualified leads per month at $X CPL without managing the funnel."
     Verbs: Receive, Hit, Cut, Stop, Eliminate, Unlock, Get, Automate, See, Free up.
   Prefix with 💥 or 🚀 either way.

7. REFRAME LINE — "Most [segment] don't have a [obvious problem] — they have a [real
   underlying problem]." Pattern interrupt. Repositions the host as a diagnostician.

8. COMPETITIVE URGENCY CLOSE — ADAPTS to host business_model.
   • Educational: "The [segment] winning in 2026 aren't doing [old approach]… They're
     building [new system] that generates [result] on repeat."
   • Delivery: "The [segment] winning in 2026 aren't running [X] in-house anymore…
     They're using [host] / handing it to [host] / plugging into [host] for [outcome]."
   No false scarcity in either shape.

9. P.S. REPLAY HOOK — "P.S. Want the replay? Just register through the official page."
   Removes FOMO friction. Increases total registrations.

---
Note: This fallback is used only when the Deal Forge Supabase format brain is unreachable.
The live format brain takes precedence and may differ from this fallback.
